# Flask REST MongoDB

## Descriptions

Simple REST API to manage User model

<br>

### Installation

Clone repo:
```shell
$ git clone https://github.com/bojandro/flast-rest-mongo.git
```

<br>

Go to project dir:
```shell
$ cd flask-rest-mongo
```

<br>

Run docker-compose:
```shell
$ docker-compose up --build
```

<br>

### Docs

`Swagger` API docs are available by going to `localhost:5000`
